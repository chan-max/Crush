
# 什么是Crush

Crush 是一个类vue的javascript框架，专注于视图层，相比于vue，扩展了css语言，使得可以用更加抽象，简介，优雅的模板语言来描述你的页面

##### 第一个例子

来想一下如何写一个长宽相等的div

```html
    <style>
        div{
            width:x;
            height:x;
        }
    </style>
    <div>
    </div>
    <script>
        var app = {
            init(){
                var {
                    x,
                    setX
                } = useNumber(100)
            }
        }
    </script>
```
