# 元素过渡与动画

## 引导 

元素的过渡主要是由两种方式实现，一种是在进入与离开时添加相应的class，添加的类名如下


- transition-enter
- transition-enter-from
- transition-enter-to

- transition-leave
- transition-leave-from
- transition-leave-to

也可以通过自定义的名称使用对应的类名



## 关键帧过渡

动画过渡的另一种实现方式是让元素进入和离开时执行一个关键帧动画


库内部提供了如下方法用于实现元素的过渡与动画效果

- transtion 组件
- transtion 指令
- transtion-group 组件
- transtion-group 指令

下面为我们逐一讲解

## transtion 组件

### transtion props

- type : css | animate
- name : 用于自定义class的名称，默认为 transtion
- 

```
	<transtion type="css">
	
	</transtion>
```
